## Description

Tavern is an interactive learning platform which gives you feeling of accomplishment with RPG-like progression system!

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
